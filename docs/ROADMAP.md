# Roadmap del Proyecto

## Tareas Completadas
- [x] Integración de sección "Preguntas Frecuentes" (FAQ) en la página principal (español e inglés).
- [x] Corrección estética del logotipo principal del Hero (nitidez, contraste, y adición de luz cian en lugar de naranja).

## Próximas Mejoras Prioritarias
